### 3. `technical_analysis_with_ai.py` — 기술적 분석 고도화 (7번)

**요청**: 시장 구매자 입장에서 진정으로 도움이 되도록 전면 개선

**추가된 기능**:

| 기능 | 설명 |
|------|------|
| ADX (Average Directional Index) | `calc_adx()` — Wilder's EWM 방식, 추세 강도 수치화 |
| OBV 다이버전스 감지 | `calc_obv()` — 가격 방향 vs OBV 방향 불일치 → 추세 반전 조기 신호 |
| 주간 피벗 포인트 | `calc_pivot_weekly()` — 지난 주 H/L/C 기반 (일봉 피벗 → 주봉으로 변경) |
| 거래량 프로파일 POC | `calc_volume_profile()` — 최대 거래량 가격대 (Point of Control) |
| 종합 매수/매도 점수 | `calc_composite_score()` — 추세(-3~+3) + 모멘텀(-3~+3) + 거래량(-1~+1) = 합계(-7~+7) |

**점수 레이블**:
```
≥+5: 강한매수  +3~4: 매수  +1~2: 약매수
-1~0: 중립
-2~-3: 약매도  -4~-5: 매도  ≤-6: 강한매도
```

### 4. `support_resistance.py` — 지지·저항선 분석 고도화 (8번)

**요청**: 더 실용적인 지지·저항 분석

**추가된 기능**:

| 기능 | 설명 |
|------|------|
| 피벗 감지 윈도우 | `window=5` → `window=10` (노이즈 감소) |
| 터치 횟수 카운트 | `count_touches()` — 최근 252일 내 해당 레벨 접근 횟수 |
| 강도 등급 | `rate_strength()` — ★/★★/★★★ (터치 횟수 + 최근성 점수) |
| 심리적 가격대 | `find_round_numbers()` — 자산 클래스별 라운드 넘버 탐지 |

**라운드 넘버 스텝**:
```python
if curr >= 100000: steps = [10000, 50000]    # 삼성전자 등
elif curr >= 10000: steps = [1000, 5000]
elif curr >= 1000:  steps = [50, 100]        # USD/KRW
elif curr >= 100:   steps = [10, 25, 50]     # SPY/QQQ
elif curr >= 10:    steps = [5, 10]
else:               steps = [0.5, 1]
```
